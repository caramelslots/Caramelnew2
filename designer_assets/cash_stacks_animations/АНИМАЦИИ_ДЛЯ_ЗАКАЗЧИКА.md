# Анимации в слоте Cash Stacks — кратко для заказчика

## Главное

В игре **два вида движения**:

1. **Движение барабанов** (крутятся, останавливаются) — делает **программа**. Отдельно рисовать «анимацию вращения» для каждого символа не нужно — на барабане показывается обычная картинка символа.

2. **«Живые» эффекты** (выигрыш символа, взрыв Wild, фон, Big Win, переход в бонусы) — делаются в программе **Spine**. Это отдельные анимированные файлы от дизайнера/аниматора.

## Список анимаций (Spine), которые нужны в игре

| Элемент                         | Какие анимации нужны                                                                                                         | Когда игрок это видит                                                 |
| ------------------------------- | ---------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------- |
| **H1** (высокий символ 1)       | Выигрыш: `h1`                                                                                                                | Символ в выигрышной комбинации                                        |
| **H2**                          | Выигрыш: `h2`                                                                                                                | То же                                                                 |
| **H3**                          | Выигрыш: `h3`                                                                                                                | То же                                                                 |
| **H4**                          | Выигрыш: `h4`                                                                                                                | То же                                                                 |
| **L1** (низкий символ 1)        | Выигрыш: `l1`                                                                                                                | То же                                                                 |
| **L2**                          | Выигрыш: `l2`                                                                                                                | То же                                                                 |
| **L3**                          | Выигрыш: `l3`                                                                                                                | То же                                                                 |
| **L4**                          | Выигрыш: `l4`                                                                                                                | То же                                                                 |
| **Wild (W)**                    | Приземление: `wild_dynamite_land`; выигрыш: `wild_dynamite` (с моментом «взрыва» под звук)                                   | Wild выпал на барабан / участвует в выигрыше                          |
| **Bonus (B)**                   | **Нужно разработать** (сейчас анимации нет)                                                                                  | Попадание scatter, триггер бонуса — желательно отдельная win-анимация |
| **Mystery (M)**                 | Раскрытие: `high_multiplier_pay` / `mid_multiplier_pay` / `low_multiplier_pay` (зависит от того, во что превращается символ) | Секретный символ «?» открывается в обычный символ                     |
| **Подсветка выигрыша**          | Рамка вокруг символа: `payframe` (зациклена)                                                                                 | Любой выигрышный символ на поле                                       |
| **Фон — обычная игра**          | `idle` + `dust` (оба на повторе)                                                                                             | Весь базовый режим                                                    |
| **Фон — фриспины**              | `idle` + `dust` (оба на повторе)                                                                                             | Режим Free Spins                                                      |
| **Экран загрузки**              | `title_screen` (на повторе)                                                                                                  | Пока игра грузится                                                    |
| **Переход между режимами**      | `animation` (один раз)                                                                                                       | Вход в фриспины и выход обратно                                       |
| **Свечение барабанов в бонусе** | `reelhouse_glow_start` → `reelhouse_glow_idle` → `reelhouse_glow_exit`                                                       | Free Spins: появление, горение, исчезновение                          |
| **BIG WIN**                     | `big_win_intro` → `big_win_idle` → `big_win_exit`                                                                            | Крупный выигрыш (уровень Big)                                         |
| **SUPER WIN**                   | `super_win_intro` → `super_win_idle` → `super_win_exit`                                                                      | Ещё крупнее                                                           |
| **MEGA WIN**                    | `mega_win_intro` → `mega_win_idle` → `mega_win_exit`                                                                         | Ещё крупнее                                                           |
| **EPIC WIN**                    | `epic_win_intro` → `epic_win_idle` → `epic_win_exit`                                                                         | Ещё крупнее                                                           |
| **MAX WIN**                     | `max_win_intro` → `max_win_idle` → `max_win_exit`                                                                            | Максимальный уровень выигрыша                                         |
| **Экран старта фриспинов**      | `intro` → `idle` (фон панели)                                                                                                | «Вы выиграли N фриспинов»                                             |
| **Число фриспинов**             | `intro` → `idle` (слот под цифру N)                                                                                          | То же окно, анимация числа                                            |
| **Экран итога фриспинов**       | `intro` → `idle` (фон панели)                                                                                                | Конец бонуса, итоговая сумма                                          |
| **Число итогового выигрыша**    | `intro` → `idle` (слот под сумму)                                                                                            | То же окно, анимация суммы                                            |
