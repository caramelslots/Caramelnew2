# Визуальные ассеты Cash Stacks (полный каталог)

Здесь перечислены **все картинки и текстуры**, которые видит игрок: и **без анимации** (спрайты на барабане, рамки, тексты), и **части Spine** (текстурные атласы + скелеты в `sample/spines/`).

| Что | Где |
|-----|-----|
| **Этот файл + `sample/`** | Все визуальные файлы для просмотра и замены |
| **Клипы анимации** (имена, когда играть) | `../cash_stacks_animations/АНИМАЦИИ_ДЛЯ_ЗАКАЗЧИКА.md` |
| **Звук** | `../cash_stacks_sounds/ЗВУКИ.md` |

> **Кнопки UI** (Spin, Turbo, меню) — без картинок, рисуются кодом.  
> **Символы B и M на барабане** — заглушки (нужен новый арт). Для **M** при раскрытии есть Spine-текстуры в `sample/spines/symbols2/`.

---

## Как устроен ассет

| Тип | Файлы | Анимация |
|-----|-------|----------|
| **Спрайт-атлас** | `*.json` + `*.png` / `*.webp` | Нет (одна картинка на кадр) |
| **Spine** | `*.atlas` + текстура + `*.json` (скелет) | Да — клипы внутри `.json`; список клипов → папка `animations` |
| **Bitmap-шрифт** | `*.xml` + `*.webp` | Нет (глифы) |
| **Монетки Big Win** | спрайт-лист `1.png`…`12.png` | Да, но через код (частицы), не Spine |

## Как открыть атлас

**Спрайты** (`01`–`07`): `*.json` + одна текстура — кадры перечислены в JSON.

**Spine**: откройте `*.json` скелет в [Spine](http://esotericsoftware.com/) вместе с `*.atlas` и текстурой. Текстура — это «картинка», из которой собран персонаж; клипы анимации — в том же проекте Spine.

## Примечания для дизайнера

1. **Символ на барабане** = спрайт (`01_…`); **выигрыш того же символа** = Spine (`spines/symbols/`).
2. При замене спрайтов сохраняйте **имена кадров** в атласе (`h1.webp`, …).
3. При замене Spine сохраняйте **имена клипов** в `.json` — см. `../cash_stacks_animations/`.
4. Текстовые PNG — с прозрачным фоном (RGBA).
